Interpretation Guide

Durbin-Watson: Tests serial correlation in residuals. Values near 2.0 = no autocorrelation. <1.5 or >2.5 = concern.
Breusch-Pagan: Tests heteroscedasticity. Low p-value = residual variance is not constant → use robust SE (HC1).
Jarque-Bera: Tests normality of residuals. Low p-value = non-normal (common in finance — fat tails).
Adjusted R²: Penalizes for number of regressors. Closer to R² = good; much lower = overfitting.
F-statistic: Joint significance — tests whether ALL factors together are significant. p≈0 = model is meaningful.
Bootstrap CI: If 95% CI does not contain zero, the LASSO coefficient is statistically significant at 5% level.
